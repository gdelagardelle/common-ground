# TestFlight — What to Test (copy-paste)

## Beta App Description

Common Ground Co-Parent helps separated parents coordinate custody schedules, shared expenses, medical info, and court-ready exports. This beta validates core flows before App Store release.

## What to Test

Please verify:

1. **Onboarding** — Create a family and add a child
2. **Home** — Quick actions (event, expense, message) and dashboard cards
3. **Calendar** — Custody events, exchange reminders, schedule builder (More → custody tools)
4. **Children** — Child profile, medical, school, expenses tabs
5. **Messages** — Co-parent thread (demo data included if using fresh install)
6. **More** — Court export PDF, invite co-parent code, optional Calendar sync and Health import
7. **Privacy** — Face ID lock toggle under More → Privacy & Security

## Known limitations

- iCloud sync is off by default; enable under More → iCloud Sync and restart
- On-device AI requires iOS 26 + Apple Intelligence
- Live Activities and Health import require a physical device
- Co-parent join across devices needs iCloud sync on both phones
