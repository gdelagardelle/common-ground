## Description

Common Ground Co-Parent is the shared home for raising children across households — built with the polish and privacy you expect from a native Apple app.

**Coordinate without conflict**
- Shared custody calendar with schedule builder (week on/off, 2-2-3, alternating weekends)
- Custody exchange reminders and Live Activities
- Two-way Apple Calendar sync

**Everything about your child, in one place**
- Medical records, medications, and growth charts from Apple Health
- School info, expenses with settle-up, documents, and milestones
- Secure messaging between co-parents

**Private by design**
- Face ID / Touch ID lock
- On-device AI assistant powered by Apple Intelligence (iOS 26+)
- Court-ready PDF export with integrity hash
- Local-first storage — your data stays yours

**Built for real co-parents**
Whether you're navigating a divorce, separation, blended family, or shared custody — Common Ground keeps both households aligned around what matters: your children.

## Review Notes

Demo account not required — app works fully offline after onboarding.

To test co-parent join on one device:
1. Complete onboarding on Device A (or Simulator A with data)
2. Note the 8-character family code from More → Invite Co-Parent
3. Delete app, reinstall, tap "Join with Family Code" on welcome screen
4. Enter code — requires same SwiftData store; for cross-device, enable iCloud sync

Health import requires a physical device with Health app data.
