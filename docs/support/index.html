Getting started

  • Complete onboarding to create a family and add a child.
  • Use More for Calendar sync, Health import, court export, and co-parent invite.
  • Enable Face ID lock under More → Privacy & Security.

Co-parent invite

Share your 8-character family code from More → Invite Co-Parent. The other parent installs the app and taps Join with Family Code on the welcome screen. Cross-device sharing requires iCloud sync to be enabled on both devices.

iCloud sync

Turn on sync under More → iCloud Sync, then restart the app. Requires an Apple ID signed into iCloud and CloudKit capability enabled in your Xcode build for development builds.

TestFlight

If you received a TestFlight invite, install the TestFlight app from the App Store first, then accept the invite link from your email.
